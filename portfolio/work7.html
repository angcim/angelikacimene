In Motion Exercise

Winter 2017

Challenge: Create an animated composition by using code. This composition must have a timed factor, meaning that there should be elements that appear, change, and move in different times.

Solution: For this exercise, I recreated Tetris with the program Processing. The reason why I chose Tetris was because it was challenging way to code time elements in Processing. For my process, I looked at Tetris images through Google and recreated it with Processing. I coded rectangle and ellipse functions to design it. In order to animate the blocks going down, I used float functions to change the blocks in the small preview screen and to control how fast and slow the blocks are moving in Tetris. With these functions, it gives my design a nice flow and similarity of the game Tetris. The color choices I chose were based on the actual classic game. I chose these colors to truly represent that game to make it recognizable to users. With such recognition, it gives the viewers a nice nostalgic effect.

Tools:
Processing
